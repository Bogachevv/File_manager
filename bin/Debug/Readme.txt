Left panel for local disk.
Right panel for server(connect by FTP).
If you want to change FTP connection settings you should lock at "Connection" menu:
	If you press "info" button, you see connection info.
	If you press "set" button, you can change connection data.
	If you press "save" button, you can save connection data to config file.
To drop config:
	1)Open cmd
	2)Run command "cd <way>" where <way> - way to filemanager.exe
	3)Run command file_manager.exe ftp:<ip> <login> <password> <localHome> <serverHome>
	4)Press "connection"->"save" buttons.
